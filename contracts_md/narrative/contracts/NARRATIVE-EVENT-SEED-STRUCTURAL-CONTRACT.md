🟠 NARRATIVE EVENT SEED
Structural Contract v2.0
(Post-Atomicity / Pre-NCA / Ontology-Pure)

I. ROLE DEFINITION
A Narrative Event Seed captures:
A single, discrete, observable occurrence.
It describes:
What happened.
It does not describe:
* Structural configuration
* Structural asymmetry
* Media framing
* Symbolic interpretation
* Outcome
* Persistence
* Escalation
* Causality
It records one event only.

II. ARCHITECTURAL POSITION (UPDATED)
Pipeline position:
Source Material→ REC generation→ Atomicity Enforcement (Global Split Doctrine)→ Narrative Event Seed Typing (this contract)→ NCA→ META / SANTA
Narrative Event Seeds are:
Post-split atomic units only.

III. ATOMICITY LOCK (NEW EXPLICIT STATEMENT)
Narrative Event Seeds:
* MUST originate from a single Atomic Unit (AU).
* MUST represent exactly one ontological component.
* MUST NOT contain composite material.
* MUST NOT require internal splitting.
Composite rejection occurs at Atomicity Enforcement, before seed typing.
If composite material appears here:
→ Upstream violation→ Reject unit
No splitting may occur inside this contract.

IV. STRUCTURAL FORMAT
Each Narrative Event Seed must follow:
SEED: [Event Title]

Event Description:
[Single neutral declarative sentence describing what occurred]
Optional:
Time Marker (if directly observable)Location (if directly observable)
No other fields permitted.

V. HARD CONSTRAINTS
The Event Description must:
✔ Be declarative✔ Be factual✔ Be observable✔ Be time-bound✔ Be actor-attributable✔ Avoid interpretation✔ Avoid escalation language✔ Avoid causal phrasing

VI. PROHIBITED LANGUAGE
Disallowed:
* “signaled”
* “sparked”
* “marked the beginning”
* “historic”
* “turned the tide”
* “changed the trajectory”
* “created tension”
* “escalated”
* “led to”
* “resulting in”
* “because”
* “therefore”
Narrative Event Seeds must not imply:
MeaningImpactTrajectoryResolution

VII. SINGLE-OCCURRENCE RULE
A Narrative Event Seed must describe:
Exactly one discrete occurrence.
It may not:
* Combine multiple actions
* Aggregate a sequence
* Summarize a trend
* Imply duration
Incorrect:Player argued with coach and later skipped media availability.
Correct:Two separate Narrative Event Seeds.

VIII. NO STRUCTURAL RECONFIGURATION CLAIMS
Narrative Event Seeds may describe:
An action.
They may not describe:
Structural interpretation of the action.
Example:
Allowed:Head coach announced a quarterback change.
Not allowed:The quarterback change altered the power structure of the offense.
Structural interpretation belongs elsewhere.

IX. NO MEDIA FRAMING
If the unit describes:
Public labelingReputation framingRhetorical positioning
It is not a Narrative Event Seed.
It is Media Context.

X. IDENTITY ABSTRACTION
All proper names must be replaced with role identifiers:
Head CoachVeteran QuarterbackRookie QuarterbackExecutiveFranchiseLeague
Seeds must remain simulation-portable.

XI. FAILURE CONDITIONS
Reject if:
* Composite material present
* More than one occurrence described
* Contains causal language
* Contains interpretation
* Contains escalation language
* Contains framing
* Contains structural asymmetry
* Contains structural reconfiguration analysis

XII. IDENTITY LOCK
Narrative Event Seeds are:
* Atomic
* Post-split
* Single-occurrence
* Non-interpretive
* Pre-classification
* Pre-escalation
* Pre-canonical
They capture that something happened.
Nothing more.
