ORGANIZATION STATEMENT BOUNDARY DOCTRINE
Event vs Media Context Routing Rule — v1.0

I. PURPOSE
This doctrine resolves routing ambiguity for organization-issued public statements.
It ensures:
* Event lane models structural reality only
* Media Context lane absorbs interpretive framing
* EMI speech-act logic remains deterministic
* Canonical ledger remains structure-bound
* No rhetorical pollution of Event objects

II. DEFINITIONS
Organization StatementA publicly attributable communication issued by an organization, team, league, front office, or executive actor.
Structural MutationA completed, enacted change affecting:
* Personnel status
* Authority distribution
* Policy
* Contractual condition
* Role assignment
* Suspension
* Termination
* Structural resource allocation
* Governance decision
Interpretive FramingExpressions of:
* Belief
* Confidence
* Opinion
* Justification
* Strategy description (without enacted change)
* Defense
* Narrative positioning
* Emotional stance
* Evaluation

III. ROUTING RULE
1️⃣ Event Lane Eligibility
An organization statement SHALL route to Event lane if and only if:
* It asserts a completed structural mutation.
* The action is enacted, not contemplated.
* The mutation is publicly legible.
* The actor is deterministically attributable.
* The mutation satisfies EMI occurrence gate.
Examples (Event):
✔ “The team has fired the offensive coordinator.”✔ “The league suspended the player.”✔ “Management reassigned play-calling duties.”✔ “Ownership approved the trade.”

2️⃣ Media Context Lane Eligibility
An organization statement SHALL route to Media Context lane if:
* It expresses belief, confidence, or opinion.
* It frames interpretation of events.
* It justifies prior actions.
* It defends or critiques narrative.
* It signals intent without enacted change.
* It describes strategy without structural mutation.
Examples (Media Context):
✖ “We believe in our quarterback.”✖ “Management remains confident in the staff.”✖ “Ownership views this as a long-term rebuild.”✖ “The team feels the criticism is unfair.”

IV. NO INFERENCE RULE
EMI SHALL NOT infer structural mutation from interpretive language.
If structural change is not explicitly asserted in the statement:
→ Route to Media Context.

V. SPLIT RULE
If a statement contains:
* Interpretive framingand
* A completed structural mutation
Then:
* Extract structural mutation as Event
* Route full statement to Media Context lane unchanged
No mutation of source.No stripping.Parallel fork allowed.

VI. ONTOLOGICAL ALIGNMENT
This doctrine preserves:
* Ledger mutation definition of Event
* Separation between structure and rhetoric
* Canonical immutability
* Media Context as framing layer
* Narrative engine as interpretive layer
Event lane represents structural reality.Media Context lane represents communicative positioning.

VII. ENFORCEMENT LOCATION
This rule SHALL be enforced at EMI during occurrence evaluation.
Media Context routing SHALL occur when structural mutation condition fails.

VIII. LOCK STATEMENT
Under this doctrine:
* Completed structural action → Event
* Interpretive organizational framing → Media Context
No organizational belief statement may enter Event lane without structural mutation.
