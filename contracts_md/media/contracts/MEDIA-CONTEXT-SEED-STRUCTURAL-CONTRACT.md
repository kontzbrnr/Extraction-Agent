🟣 MEDIA CONTEXT SEED
Structural Contract v2.0
(Post-Atomicity / Discourse-Only / Ontology-Pure)

I. ROLE DEFINITION
A Media Context Seed captures:
Public discourse framing about events, actors, conditions, or structures.
It captures:
Language about reality.
It does not capture:
* The event itself
* The structural configuration
* The pressure asymmetry
* The symbolic physical act
* Procedural change
* Official designation
Media Context describes framing, labeling, or rhetorical positioning.

II. ARCHITECTURAL POSITION (UPDATED)
Pipeline position:
Source Material→ REC generation→ Atomicity Enforcement (Global Split Doctrine)→ Media Context Seed Typing (this contract)→ Context Registry
Media Context Seeds are:
Post-split atomic units only.
Composite rejection occurs pre-typing.
No splitting may occur within this contract.

III. CORE STRUCTURAL CRITERIA
A unit qualifies as Media Context only if:
* It describes public language, labeling, or framing.
* It does not describe a discrete action.
* It does not describe structural configuration.
* It does not describe unresolved asymmetry.
* It can be removed without altering factual state.
If removal changes factual state → not Media Context.

IV. ORGANIZATION STATEMENT BOUNDARY RULE (NEW)
An organization statement qualifies as Media Context only if:
It contains framing language without action claims.
Allowed example:
The franchise described the quarterback as “the future of the organization.”
Not allowed:
The franchise announced a quarterback change.
The second contains an action claim and must trigger EMI.
If an organization statement contains:
* An event verb
* An official designation
* A procedural action
→ It is not Media Context.

V. EVENT VERB TRIGGER RULE (NEW CLARIFICATION)
If a unit contains an event verb indicating:
* Hiring
* Firing
* Signing
* Trading
* Drafting
* Suspending
* Promoting
* Demoting
* Announcing
* Designating
* Activating
* Placing
EMI must activate.
Media Context cannot contain event verbs.
Event verb presence → pre-typing reclassification.

VI. WHAT MEDIA CONTEXT MAY DESCRIBE
Allowed surfaces:
* Reputation framing
* Narrative labeling
* Redemption framing
* Blame attribution
* Emotional crowd framing
* Media storyline emphasis
* Descriptive metaphors
* Rhetorical positioning
Example:
Analysts described the defense as undisciplined.
This does not alter structure.

VII. WHAT MEDIA CONTEXT MAY NOT DESCRIBE
Prohibited:
* Discrete action
* Official announcement of change
* Role reassignment
* Medical designation
* Structural configuration
* Structural asymmetry
* Symbolic physical act
Important distinction:
Event:Team buried a game ball at midfield.
Media Context:Commentators framed the act as a statement to the league.
Only the second qualifies.

VIII. STRUCTURAL FORMAT
Each Media Context Seed must follow:
SEED: [Framing Descriptor Title]

Context Description:
[Single neutral declarative sentence describing framing or labeling]
Optional:
Source Type (media / crowd / analyst / organization statement)Time Marker
No interpretation beyond description of framing.

IX. DISCOURSE TEST
Litmus test:
If all media commentary disappeared, would factual state change?
If yes → not Media Context.If no → Media Context.

X. SINGLE-FRAME RULE
One Media Context Seed = one framing construct.
Incorrect:Media described the team as rebuilding while questioning coach authority.
Correct:Two separate Media Context Seeds.

XI. ATOMICITY LOCK
Media Context Seeds:
* Must originate from a single Atomic Unit (AU).
* Must not contain composite material.
* Must not require splitting.
Composite handling belongs to Atomicity Enforcement only.

XII. PROHIBITED LANGUAGE
Media Context Seeds may not include:
* causing
* leading to
* resulting in
* therefore
* which created
* structural shift
* officially
* designated
Event language → EMI.Structural configuration → Structural Environment.Asymmetry language → Pressure.

XIII. IDENTITY ABSTRACTION
All proper names must be replaced with role identifiers.
Seeds must remain simulation-portable.

XIV. FAILURE CONDITIONS
Reject if:
* Contains event verb
* Contains structural configuration
* Contains asymmetry
* Contains official designation
* Contains composite material
* Contains symbolic physical act
* Contains interpretation beyond framing

XV. IDENTITY LOCK
Media Context Seeds are:
* Atomic
* Post-split
* Discourse-only
* Non-event
* Non-structural
* Non-pressure
* Non-mutative
They describe language about reality.
They do not describe reality itself.
